# Calculadora de Consumo Elétrico

📝 Objetivo:
Programa criado em python para calcular o consumo elétrico mensal e o valor gasto.

📊 Fórmulas utilizadas:
Duas fórmulas para o cálculo de wats mensais e o custo mensal obtido com base no valor de kwh de São Paulo (R$ 0,85):
consumo_mensal = (potencia_watts * tempo_horas * 30) / 1000
consumo_mensal_dinheiro = float(consumo_mensal * 0.85)

✅ Execução do programa é simples, basta inserir o nome do produto elétrico, o wats dele e a quantidade de horas média de utilização.
